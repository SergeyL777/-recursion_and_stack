## 1. Рекурсивное вычисление факториала

Функция `factorial(n)` вычисляет факториал числа $n$ с использованием рекурсии.

**Особенности:**
* Базовый случай:* $n = 0$ или $n = 1$ → возвращает $1$.
*Рекурсивный случай:* $n! = n \times (n-1)!$.

## 2. Рекурсивное вычисление суммы элементов списка

Функция `sum_list(lst)` рекурсивно вычисляет сумму всех элементов в списке.

**Особенности:**
*Базовый случай:* пустой список → возвращает $0$.
*Рекурсивный случай:* первый элемент + сумма остальных элементов.

## 3. Бинарный поиск с рекурсией

Функция `binary_search(arr, target, low=0, high=None)` выполняет бинарный поиск элемента в отсортированном списке.

**Возвращает:**
* индекс найденного элемента;
* $-1$, если элемент не найден.

**Алгоритм:**
1. Вычисляется средний индекс.
2. Если элемент найден — возвращается индекс.
3. Если средний элемент больше искомого — поиск в левой половине.
4. Если средний элемент меньше искомого — поиск в правой половине.

## 4. Класс Stack

Класс `Stack` реализует структуру данных «стек» с основными операциями:

* `push(item)` — добавление элемента на вершину стека;
* `pop()` — удаление и возврат элемента с вершины стека (вызывает `IndexError` для пустого стека);
* `is_empty()` — проверка, пуст ли стек;
* `peek()` — возврат элемента с вершины стека без удаления (вызывает `IndexError` для пустого стека);
* `size()` — возвращает количество элементов в стеке.
